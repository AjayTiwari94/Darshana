  🚀 QUICK START (3 SIMPLE STEPS)
═══════════════════════════════════════════════════════════════

STEP 1: First Time Setup
   └─> Double-click: FIRST_TIME_SETUP.bat

STEP 2: Add Gemini API Key
   └─> Open: ai-service\.env
   └─> Add: GEMINI_API_KEY=your_actual_key_here
   └─> Get key from: https://makersuite.google.com/app/apikey

STEP 3: Start Everything
   └─> Double-click: START_ALL.bat
   └─> Wait 15 seconds
   └─> Open browser: http://localhost:3000

   ❌ NEVER run "python app.py" directly without venv


═══════════════════════════════════════════════════════════════
  🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

   Problem: Services won't start
   Solution: Run as Administrator

   Problem: Narad AI not responding
   Solution: Check GEMINI_API_KEY in ai-service\.env

   Problem: Port already in use
   Solution: STOP_ALL.bat, then START_ALL.bat

   Problem: MongoDB error
   Solution: net start MongoDB
